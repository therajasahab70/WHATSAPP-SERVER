# WhatsApp Automatic QR Code Sender

## Deployment steps for Render:
1. In teeno files ko apne GitHub Repo me upload karein.
2. Render.com par New Web Service banayein.
3. Build Command: npm install
4. Start Command: npm start
5. Render Dashboard me jaakar ek Persistent Disk jarur attach karein taaki auth_info_baileys folder delete na ho (varna har 20 min me scan karna padega).
6. URL kholte hi QR screen par display ho jayega, mobile se Link Device karke direct scan karein!
